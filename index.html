Installation

This gem is currently maintined for use with Rails 3.2 and 4.0-rc1. To install the gem, add the following line to your Gemfile: gem 'batch-rails-4', '~> 0.2.3' and then run $ bundle install. There are a few development dependencies: activesupport, sass-rails, tzinfo and runtime dependencies: railties.

Usage

To insert an icon, use the following css format in your html: <i class="batch batch-#{icon-name}"></i>. You can view a full list of icons in the cheatsheet section.
